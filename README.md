# Commute Architect

Commute Architect is a tool to calculate and compare commute options based on time, cost, and carbon footprint.

## Status

The application is currently in a "Private Beta" phase.

## Development

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```
